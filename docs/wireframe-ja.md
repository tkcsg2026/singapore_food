## 1. サイト構成概要

| URL | ページ名 | 概要 |
|-----|---------|------|
| `/` | ホームページ | コンバージョン重視の業界ハブ |

## 2. ホームページ

### 目的
即座に利用可能。高速検索アクセス。高い信頼性。

### ヘッダー（スティッキー）
```
┌────────────────────────────────────────────────────────┐
│  [食] F&Bポータル    │ サプライヤー │ マーケット │ ニュース │ [ログイン] │
└────────────────────────────────────────────────────────┘
```

- **左**: ロゴ（食 アイコン + F&Bポータル）
- **中央**: ナビゲーションリンク
- **右**: ログイン/ユーザーメニュー
- **モバイル**: ハンバーガーメニュー + 固定ボトムナビゲーション

### ヒーローセクション
```
┌──────────────────────────────────────────┐
│  [背景画像: hero-bg.jpg]                  │

│                                          │
│  シンガポール                              │
│  F&B サプライヤー＆                        │
│  シェフネットワーク                        │
│                                          │
│  ┌──────────────────────────────────┐    │
│  │ [全カテゴリー ▼] [キーワード検索] [検索]│    │
│  └──────────────────────────────────┘    │
└──────────────────────────────────────────┘
```

- **タイトル**: 「シンガポール F&B サプライヤー＆シェフネットワーク」

## 3. サプライヤーディレクトリ

### URL: `/suppliers`

### レイアウト
- **デスクトップ**: 左=フィルターサイドバー / 右=サプライヤーカードグリッド
- **モバイル**: フィルタースライドアウトパネル

### フィルター項目
| フィルター | タイプ | 詳細 |
|-----------|-------|------|
| カテゴリー | マルチセレクト | 海鮮・肉類・野菜 等 |
| 少量対応 | トグル | タグフィルター |
| 日本語対応 | トグル | タグフィルター |
| ハラール | トグル | タグフィルター |
| エリア | マルチセレクト | 中央・東部・西部・北部・南部 |

### サプライヤーカード
```
┌────────────────────────┐
│ [ロゴ] ビジネス名       │
│        📍 エリア        │
│                        │
│ 説明テキスト (120文字)  │
│                        │
│ [タグ1] [タグ2] [タグ3] │
│                        │
│ [詳細を見る] [WhatsApp] │
└────────────────────────┘
```

### WhatsApp URL形式
```
https://wa.me/65XXXXXXXX?text=Hello, this is [Company], [User], interested in [Supplier].
```

```
┌──────────┐  ┌──────────┐  ┌──────────┐
```

- **固定ボトムバー**: [WhatsApp連絡] ボタン（高さ60px以上）

### フィルター

| コンディション | 新品同様 / 良好 / 使用感あり / 要修理 |
| 並び替え | 新着順 / 価格安い順 / 価格高い順 |

### グリッドビュー
- **モバイル**: 2列
- **デスクトップ**: 4列

```
│               │
│ S$2,500 (太字) │

```

## 11. 管理者ダッシュボード

### URL: `/admin-dashboard`

### アクセス制御
- `profile.role === "admin"` のユーザーのみアクセス可能

### パネル構成

#### 1. サプライヤー管理
- 一覧表示（ロゴ・名前・カテゴリー・閲覧数）
- 追加・編集・削除
- フォーム: 名前 / スラッグ / カテゴリー / エリア / タグ / WhatsApp / ロゴ / 認証 / 概要

#### 2. マーケットプレイス承認
- 審査待ちアイテム一覧
- [承認] [却下（理由入力）] ボタン
- 却下理由はユーザーダッシュボードに表示

#### 3. ニュース管理
- 記事の作成・編集・削除
- 公開/下書き切り替え
- フォーム: タイトル / スラッグ / 画像 / カテゴリー / 著者 / 内容 / 抜粋

#### 4. カテゴリー管理
- サプライヤー / マーケットプレイス / ニュース のカテゴリーを管理
- 追加・削除

#### 5. QRリダイレクト管理
- `/go` のリダイレクト先URLを設定
- QRコードは常に `/go` を指す → 印刷変更不要

#### 6. レポート管理
- ユーザーからの報告一覧
- [確認済み] [却下] アクション

#### 7. 統計スナップショット
- サプライヤー数
- マーケット出品数
- 総閲覧数
- 注目サプライヤー数
- 閲覧数トップ5

## 12. QRリダイレクト

### URL: `/go`

### 仕組み
1. QRコードは常に `yourdomain.com/go` を指す
2. アクセス時にデータベースから `qr_redirect_url` を取得
3. 設定されたURLにリダイレクト

### 管理者が変更可能なリダイレクト先
- 人気サプライヤーページ
- カテゴリーページ
- キャンペーンページ
- 外部URL

### メリット
- QRコードの再印刷が不要
- リアルタイムで変更可能
