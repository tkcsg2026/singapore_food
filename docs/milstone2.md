## 4. 管理者ダッシュボード（Admin Dashboard）

管理者専用のダッシュボードで、コンテンツ・設定・承認を一括管理できます。

### ダッシュボード構成

- **サプライヤー管理:** CRUD、プラン設定（Basic / Standard / Premium）、ロゴ・カテゴリー・エリア・WhatsApp 等。
- **Approvals:** マーケットプレイス出品の承認・却下。
- **News:** ニュース記事の作成・編集。
- **Categories:** カテゴリーの追加・削除。
- **QR Redirect:** リダイレクト先 URL の設定。
- **Reports:** 報告一覧・対応。
- **Analytics:** プラン別サプライヤー数などの集計。

![管理者ダッシュボード — サイドバー・サプライヤー管理・Premium/Standard/Basic表示](./docs/milestone2/image (7).png)

## 5. QRリダイレクト（QR Redirect）

短縮URL `/go` で設定されたURLへリダイレクトする機能を実装しています。

### 管理画面での設定

管理者ダッシュボードの「QR Redirect」タブで、リダイレクト先（例: `/suppliers` や外部URL）を設定・保存できます。

![管理者ダッシュボード — QR Redirect タブがサイドバーに表示](./docs/milestone2/image (7).png)

### 動作

- エンドポイント: `/go`
- API で `qr_redirect_url` を取得し、同一サイトパスまたは外部 URL へリダイレクト。
- QRコードを `/go` に張り、印刷物・名刺等で利用可能。

**実装状況:** ✅ 完全実装（/go ページ・設定API・管理画面）

### 言語切り替えUI

ヘッダー右側に「EN」「日本語」のトグルがあり、選択状態が視覚的に分かります。選択はローカルストレージに保存され、再訪問時も維持されます。

![ホーム（英語） — EN選択・検索バー・カテゴリーチップ](./docs/milestone2/image (11).png)

- ナビゲーション・ボタン・ラベル・プレースホルダー: 選択言語の翻訳を表示。
- カテゴリー名・エリア名: 英語時は英語、日本語時は日本語で表示。
- サプライヤー名・説明: `name` / `name_ja`、`description` / `description_ja` を言語に応じて切り替え。

## 7. 3段階プラン管理（3-Tier Plan Control）

サプライヤー向けに Basic / Standard / Premium の3プランを用意し、表示・機能を制御しています。

### プラン別の違い

- **Premium:** 最上位表示・WhatsApp表示・バッジ（星/王冠）・枠線で強調。
- **Standard:** 上位表示・WhatsApp表示・Standard バッジ。
- **Basic:** 通常掲載・カード上の WhatsApp ボタン非表示（アップグレード促進）。

### 一覧・詳細での表示

サプライヤーカードに「Premium」「Standard」バッジを表示し、Basic はバッジなしです。

![サプライヤー一覧 — Premium・Standardバッジ付きカード](./docs/milestone2/image (3).png)

![サプライヤーディレクトリ — 6件・各カードのプランバッジ](./docs/milestone2/image (5).png)

### 管理者での設定

管理者ダッシュボードのサプライヤー編集で、各サプライヤーのプラン（Basic / Standard / Premium）を設定・変更できます。一覧でもプラン別の件数が確認できます。

![管理者ダッシュボード — サプライヤーリストのPremium/Standard/Basic表示](./docs/milestone2/image (7).png)

**実装状況:** ✅ 完全実装（3プラン定義・表示制御・バッジ・管理画面での設定）

## 8. ランダム表示（Random Display）

サプライヤー一覧の表示順を「プラン優先＋日替わりランダム」で制御しています。

### 仕様

- **並び順:** Premium → Standard → Basic の順。
- **同一プラン内:** 日付ベースのシードでシャッフルし、毎日同じ順序で、日をまたぐと順序が変わるようにしています。
- **適用箇所:** ホームの「人気サプライヤー」、サプライヤーディレクトリの一覧の両方で `sortSuppliersByPlan()` を利用。

### 画面での見え方

ホームでは「Popular Suppliers」として複数サプライヤーが並び、サプライヤーページではフィルター後の結果が同様のルールで並び替えられます。スクリーンショット上は固定表示のため「ランダム」そのものは見えませんが、実装により公平なローテーションが行われています。

![ホーム — 人気サプライヤー4件の表示例](./docs/milestone2/image (3).png)

**実装状況:** ✅ 完全実装（日付シード・プラン別ソート・シャッフル・一覧・ホームへの適用）
